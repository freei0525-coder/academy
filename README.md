# 📚 학원 체크 앱 — 설치 가이드

총 3단계예요. Firebase 설정이 핵심이고, 나머지는 복붙이에요.

---

## STEP 1 — Firebase 프로젝트 만들기

1. https://console.firebase.google.com 접속
   → 구글 계정으로 로그인

2. **[프로젝트 만들기]** 클릭
   → 이름 입력 (예: hakwon-check)
   → Google 애널리틱스: "사용 설정 안 함" 선택
   → [프로젝트 만들기] 클릭
   → 완료될 때까지 기다린 후 [계속] 클릭

3. 왼쪽 메뉴에서 **[빌드 → Realtime Database]** 클릭
   → [데이터베이스 만들기] 클릭
   → 위치: "미국 (us-central1)" 선택
   → 보안 규칙: **"테스트 모드에서 시작"** 선택
   → [사용 설정] 클릭

4. 왼쪽 메뉴 상단 ⚙️ **[프로젝트 설정]** 클릭
   → 아래로 스크롤 → "내 앱" 섹션
   → **</>** (웹) 아이콘 클릭
   → 앱 닉네임 입력 (아무거나) → [앱 등록] 클릭
   → 화면에 표시되는 **firebaseConfig 코드 블록 전체를 복사**

---

## STEP 2 — index.html 에 붙여넣기

index.html 파일을 메모장(또는 텍스트 편집기)으로 열기

아래 부분을 찾아서:

```
const firebaseConfig = {
  apiKey:            "여기에_apiKey_붙여넣기",
  authDomain:        "여기에_authDomain_붙여넣기",
  databaseURL:       "여기에_databaseURL_붙여넣기",
  ...
};
```

Firebase에서 복사한 값으로 교체 후 저장

⚠️ **databaseURL이 없으면** Firebase 콘솔 → Realtime Database 화면 상단에서
   https://프로젝트명-default-rtdb.firebaseio.com 형태의 주소를 복사해서 추가

---

## STEP 3 — Netlify에 드래그앤드롭 (배포)

1. https://app.netlify.com/drop 접속
2. 이 폴더 전체를 브라우저 화면에 드래그앤드롭
3. 잠깐 기다리면 URL 자동 발급
   예: https://wonderful-name-abc123.netlify.app

4. 그 URL을 카카오톡으로 아이에게 공유!

## 앱 설치 (홈화면에 추가)

### 안드로이드 크롬
주소창 옆 ⋮ → "홈 화면에 추가" 또는 설치 배너 탭

---

## 사용법

| 역할 | 하는 일 |
|------|---------|
| 🎒 아이 | 학원 도착 후 "도착했어요" → 끝나면 "수업 끝났어요" |
| 👨 아빠 | 현황 화면에서 실시간 확인 |
| ⚙️ 관리자 | 앱 제목 3번 탭 → 스케줄 추가·수정·삭제 |

- **가족 코드**: 아이·아빠 폰에 동일한 숫자 입력 → 자동 연동
- **시간 설정**: 10분 단위 (9:00, 9:10, 9:20...)
- **알람**: 학원 시작 10분 전 자동 알림 (앱 열려있을 때)
